# Flying Bicycle Sim

A VR biking simulation using SlimeVR body trackers (via Virtual Motion Capture or VMC protocol). Developed in Godot (v4.7.dev) for the Apple Vision Pro and OpenXR (specifically Meta Quest).

## Example Images

SlimeVR Trackers are used to capture knee motion, which then translates into virtual forward motion in the simulation.
